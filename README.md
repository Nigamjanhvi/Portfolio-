# Janhvi Nigam Portfolio

A personal portfolio website built with React and Vite to showcase my projects, skills, certifications, achievements, extracurricular leadership, education, and resume.

## Features

- Responsive personal portfolio
- Project showcase with live demo and source code links
- Skills section with categorized technologies
- Certifications with preview support
- Achievements, badges, and extracurricular leadership section
- Resume section with quick navigation, preview, and download
- Contact section with email, phone, LinkedIn, and GitHub

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons
- React Scroll

## Project Structure

```text
src/

  images and badge assets
```

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```
